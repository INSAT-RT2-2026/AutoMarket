# AutoMarket

Luxury car marketplace built with PHP, Symfony, and MySQL.
https://automarket-production-ac0c.up.railway.app

## Structure

- `frontend/` - HTML/CSS/JS frontend
- `backend/` - Raw PHP API endpoints
- `symfony/` - Symfony application (framework layer)

## Symfony setup

```bash
cd symfony
composer install
```

Make sure to configure `symfony/.env` with your database URL before running.
